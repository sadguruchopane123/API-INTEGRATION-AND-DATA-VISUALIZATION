##This project is developed as part of the CODTECH Internship Task-1, where Python is used to fetch real-time weather data from the OpenWeatherMap API and visualize it using Matplotlib. The script sends an API request to retrieve forecast data such as temperature and humidity for a selected city (e.g., Pune), processes the JSON response, and converts it into graphical representations like line charts. The project uses libraries such as Requests for API integration and Matplotlib for data visualization. It can be run on platforms like VS Code, PyCharm, Jupyter Notebook, or any Python-supported terminal, making it a simple and effective example of combining API data handling with visual analytics.
